Desenvolvimento de Sistemas

O que é Desenvolvimento de Sistemas

é uma área responsável por analisar, desenvolver, analisar, projetar, implementar e atualizar sistemas de informação.

O que se estuda em Desenvolvimento de Sistemas?

Disciplinas que formam a grade curricular deste curso:

Banco de Dados

O estudante irá aprender a criar e gerenciar uma base de dados, além de desenvolver métodos para manipular e proteger as informações da maneira mais segura.

Lógica de Programação

o aluno aprenderá maneiras de projetar algoritmos, sequências de ações que seguem uma determinada ordem com a finalidade de chegar a solução de um problema.

Redes de Computadores

irá aprender tudo sobre transferência segura de dados entre redes, seus tipos e meios de transmissão, além dos conhecimentos específicos sobre conexão entre redes locais e autônomas.

Segurança de Software

O aluno a pessoa entenderá um pouco mais sobre tamanha responsabilidade e aprenderá a lidar com proteção de dados, como senhas de usuários, por exemplo, além de noções de criptografia e de políticas de segurança para software.

Software para Web

vai abordar o desenvolvimento de programas, por meio da escrita de códigos, dentro de uma linguagem de programação específica.

O que é Multimidia?

desenvolvimento de comunicação visual em meios eletrônicos, organiza e prepara arquivos digitais que podem compor telas para sistemas de comunicação para diferentes finalidades.

o que se estuda em Multimidia?

Criar e adaptar layout para diferentes plataformas do marketing digital - Tratar e editar imagens e vídeos para sites - Criar motion graphics (gráficos, ícones, textos e ilustrações em movimento) - Produzir concept art (ilustrações que servem de guia para a produção de animações)
